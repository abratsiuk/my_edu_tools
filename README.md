## Overview

MyEduTools is a small educational web application with a set of helper tools for learning and practicing English vocabulary.

The project includes:
- a word lookup page with external resources (pronunciation, dictionary, videos, mnemonic hints)
- a timer tool for tracking time spent processing words

The application is built with:
- React
- TypeScript
- Vite
- Material UI (MUI)

During development, ChatGPT was actively used as an assistant for architecture decisions, UI ideas, and implementation details.
